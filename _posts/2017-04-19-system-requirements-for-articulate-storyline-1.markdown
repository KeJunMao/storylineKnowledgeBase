## 硬件

| 名称            | 最低要求   |
| -------------  | -------------  |
| CPU | 1.0 GHz或更高处理器（32位或64位） |
| 内存 | 最低512 MB |
| 存储 | 最低250 MB |
| 显示器 | 1,024 x 768屏幕分辨率 |
| 其他 | 声卡，麦克风和USB摄像头 |

## 软件

| 名称            | 最低要求   |
| -------------  | -------------  |
| 操作系统 | Microsoft Windows XP SP3或更高版本（32位或64位） |
| .NET | 	.NET 2.0 |
| Adobe Flash Player | Adobe Flash Player 10 |

## 兼容性和集成度

| 名称            | 兼容性和集成度   |
| -------------  | -------------  |
| 导入项目 | Articulate Presenter ‘09, Articulate Quizmaker ‘09, Articulate Engage ‘09, Microsoft PowerPoint 2000 及最新版 (不兼容的 PowerPoint 版本: 64-bit, Click-to-run, Office 365.)|
| 发布到Word |	Microsoft Word 2000-2013 |
| 翻译 | Microsoft Word 2000-2013 |
